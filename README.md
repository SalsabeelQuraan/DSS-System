# DSS-System
# 📊 Decision Support System for Pharmaceutical Company

## 📌 Overview

This project presents a **Decision Support System (DSS)** designed for a pharmaceutical company to improve data-driven decision-making.
The system addresses the issue of **inconsistent and unstructured data across departments** by centralizing, cleaning, and analyzing operational data.

The DSS provides analytical insights, performance indicators, scenario analysis, and recommendations to support management decisions.

## 🎯 Objectives

* Centralize data from multiple departments
* Improve data quality through preprocessing and cleaning
* Analyze operational performance
* Support decision-making using data insights
* Provide forecasting and scenario analysis

## 🏢 System Scope

The system integrates data from the following departments:

* Production
* Sales
* Finance
* Human Resources
* Logistics
* Research & Development
* Quality Control

## ⚙️ Key Features

### 🔹 Data Integration

* Import data from Excel files
* Standardized data templates across departments

### 🔹 Data Cleaning & Preprocessing

* Handle missing values
* Remove duplicates
* Standardize formats

### 🔹 Data Analysis

* Compute key metrics and KPIs
* Evaluate departmental performance

### 🔹 Decision Support

* Perform **What-if analysis**
* Evaluate impact of different business scenarios

### 🔹 Recommendation System

* Generate insights and suggested actions

### 🔹 Forecasting

* Predict future trends based on historical data

### 🔹 Visualization

* Interactive charts and dashboards

### 🔹 Reporting

* Export results as reports (Excel / PDF)

## 🧠 Technologies Used

* **Python**
* **Pandas** – Data processing
* **NumPy** – Numerical computations
* **Matplotlib / Seaborn** – Data visualization
* **Scikit-learn** – Basic machine learning
* **SQLite / MySQL** – Data storage

## 🏗️ System Architecture

The system follows a modular architecture:

1. Data Input Layer
2. Data Processing Layer
3. Analysis Engine
4. Decision Support Module
5. Visualization & Reporting

## 📊 Example Use Cases

* Analyze company performance
* Evaluate profitability
* Forecast sales trends
* Test pricing or cost scenarios
* Support strategic planning
